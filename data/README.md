# Data Directory

This directory's contents are **not** committed to git (see root `.gitignore`)
— only the structure (via `.gitkeep`) is tracked. Raw images and annotation
exports are large/binary and belong in the dataset's **Mendeley Data** release,
not in the git history.

## Folder Structure

| Folder | Purpose |
|---|---|
| `raw/` | Original product label images (JPEG, sequentially numbered: 0001.jpeg, 0002.jpeg, ...) |
| `thumbnail/` | Resized preview images (max 250px, for quick QA browsing) |
| `label_studio/` | Label Studio export JSON + pre-annotations (generated by preprocessing pipeline) |
| `processed/` | Derived data: token-level NER (`train.json`/`val.json`), per-image KIE records (`dataset/<id>.json`), `dataset_meta.json`, `splits.json`, `statistics.json` |

## Workflow

See [scripts/preprocessing/README.md](../scripts/preprocessing/README.md) for the complete workflow:

1. **Prepare images** (`prepare_images.py`): HEIC → JPEG + thumbnails
2. **Generate pre-annotations** (`label_studio_preann.py`): OCR detection + VietOCR recognition
3. **Export from Label Studio** and **Convert** (`convert_ls_to_ner.py`): Label Studio JSON → token-level NER format

Data format reference:
- [docs/dataset-overview.md](../docs/dataset-overview.md) — Record schema and properties
